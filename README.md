# Telco Churn MLOps

Proyecto final del curso especializado en MLOps.

## Caso de negocio

Una fintech busca predecir la pérdida de clientes usando un modelo de Machine Learning y buenas prácticas MLOps.

## Flujo implementado

1. Carga del dataset.
2. Preprocesamiento de datos.
3. Entrenamiento del modelo.
4. Registro de métricas con MLflow.
5. Versionamiento con Git/GitHub.
6. Automatización con GitHub Actions.
7. API local de inferencia con Flask.
8. Prueba de endpoint con Postman.
